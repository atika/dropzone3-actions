Share with Secure Data Space 1.7

Upload a file to SSP Secure Data Space and create a share link (URL will be placed clipboard). Holding ‘Command’ (⌘) or ‘Option’ (⌥) will expire the uploaded file and its share link after 14 days, ‘Control’ (^) or ‘Option’ (⌥) will allow you to set a password for the share link. Holding ‘Shift’ (⇧) just uploads the file without sharing it. Clicking on this Action copies the latest share link to the clipboard.
Icon is owned by SSP Europe GmbH.
